在 IDEA 中使用

将以下 Catalog URL 添加到 IntelliJ IDEA，一键生成项目：

Catalog 地址

已复制!

配置步骤

1

File → New → Project → Maven Archetype

2

点击右上角 Manage Catalogs → + 添加：

Catalog File: __GITHUB_PAGES_URL__/archetype-catalog.xml Description: AI Agent Scaffold
3

选中刚添加的 Catalog，下方列表找到 ai-agent-scaffold-archetype，填写 groupId、artifactId、package，点击 Create

4

IDEA 自动生成完整的多模块 DDD 项目 🎉

Maven 命令行

mvn archetype:generate \ -DarchetypeGroupId=org.example \ -DarchetypeArtifactId=ai-agent-scaffold-archetype \ -DarchetypeVersion=1.0 \ -DarchetypeCatalog=__GITHUB_PAGES_URL__/archetype-catalog.xml \ -DgroupId=com.yourcompany \ -DartifactId=my-ai-project \ -Dversion=1.0-SNAPSHOT \ -Dpackage=com.yourcompany
